NUEVO PROYECTO OPEN SOURCE

Desarrollo Diseño Programación Comunicación
El proyecto consiste en un turnador, el cual sera mostrado en un display de 4 digitos, este mismo mostrara la hora en que un turno cambia a otro, ademas de tener comunicacion serial entre algun ordenador para poder ser controlado desde el mismo.
El PCB o Circuito impreso fue diseñado con software y grabado con cortadora lazer, desde el digrama electronico hasta el PCB en formato gerber, los diseños son 100% open hardware y estara disponible para su descarga.
La programacion fue realizada con la IDE de Arduino para la parte de electronica digital y para la interfaz de escritorio fue programada con Visual Studio.
La comunicacion entre el display y la computadora se realizó gracias al puerto serie del ordenador, utilizando las terminales de Recepción(RX) y Transmicion(TX) respectivamente.
