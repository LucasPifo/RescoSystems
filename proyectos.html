Proyectos

Nuestra empresa se enfoca en la tecnologia de vanguardia, es por eso que contamos con una gran variedad de proyectos y algunos prototipos de sistemas de interaccion Humano-Máquina los cuales pueden aplicarse a cualquier sector en el que se desempeñe.

Iot Api Rest PHP/Android

Una metodologia de desarrollo es esencial para que un sistema funcione correctamente y que cumpla los requerimientos especificos del cliente, es por eso que se utilizo la metodologia XP.
Se realizó un Web Service con PHP utilizando JSON como intercambio de información logrando asi una API REST el cual seria consumida por la APP en ANDROID STUDIO, todo esto con la tecnologia de los WEBSOCKETS para una cominucación bidireccional y asi lograr una excelente experiencia de usuario.

Modbus TCP React Native PLC S7 1200

Modbus/TCP es un protocolo de comunicación diseñado que permite a equipos industriales tales como PLCs, PC, drivers para motores y otros tipos de dispositivos físicos de entrada/salida, comunicarse sobre una red Ethernet. Fue introducido por Schneider Automation como una variante de la familia de protocolos MODBUS, ampliamente usada para la supervisión y el control de equipo de automatización. Específicamente el protocolo define el uso de mensajes MODBUS en un entorno intranet o internet usando los protocolos TCP/IP.

Trampa inteligente Biosmart

El proyecto con nombre Trampa Inteligente Biosmart dicta una solución confiable y eficiente a los problemas que se presentan tanto en el hogar como en la industria sobre el control de plagas, lo cual es de gran importancia para las empresas que majenan alimentos o quienes se encarga de brindar estos servicios. Es por ello que con el protocolo MQTT se desarrollo una trampa inteligente para la cptura de isnectos.

Silo industrial HMI React Native Bluetooth

En el proyecto siguiente se muestra como con ayuda del framework React Ntive y el uso de la tecnologia bluetooth se puede simular un panel view o mas conocido com HMI, la cual puede llegar a ser muy costosa. Se desarrolló una aplicacion para controlar el llenado de un silo, en el cual se utilizo una banda transportadora la cual mueve un contenedor que a su vez es llenado mediante un conjunto de valvulas neumaticas.

Conexión http node js con PLC S7 1200

Sistema de comunicación entre un PLC S7 1200 y un servidor web desarrollado en Node JS conectado a una base de datos en MYSQL para la manipulación de datos, se diseñó una REST API con cada una de las rutas pertinentes, asi como los controladores necesarios para la aplicación, lamparas piloto para cada estado, sensor inductivo para la deteccion de metales, cada que se detecta un metal lo ingresa en la base de datos.
