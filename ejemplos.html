  • Percepron en PHP+ 

    • Perceptron Compuerta OR/AND PHP

    El Perceptron es la neurona artificial o unidad básica de inferencia en forma de discriminador lineal, a partir de lo cual se desarrolla un algoritmo capaz de generar un criterio para seleccionar un sub-grupo a partir de un grupo de componentes más grande.

    En este caso crearemos un perceptron en PHP, pero puede ser interpretado en Javasccript o cualquier otro lenguaje de programacion.

    Lo primero que debemos hacer es declarar las entradas que tendra nuestro perceptron, como en este ejemplo emularemos la compuerta AND, utilizaremos las entradas: 

    $entradas = [[1,1,1],
            [1,0,0],
            [0,1,0],
            [0,0,0],
            ];

    La variable $entradas es una matriz que contiene en el primer indice los numeros del primer set de datos. Cada uno de esos numeros son en si "neuronas" y una fila en vertical serian las "capas".

    Sabiendo esto, en el primer set de datos tendremos que el primer valor de entrada seria "1", el segundo valor de entrada seria "1", y la salida esperada o deseada seria "1". Esto en base a la arquitectura del perceptron.

    El segundo paso seria calcular la longitud de la matriz(Esto dependeria del lenguaje de programacion), asi como la longitud de alguno de los sets de entranamiento.

    $longitud=count($entradas);
$longitud2=count($entradas[0]);

    Declarar el numero de epocas, que son el numero de iteraciones que realiza el perceptron, ademas de la condicion para que el ciclo se repita hasta encontral una respuesta en concreto. Como en el ejemplo sera con valores random.

    $epocas=0;
$ciclo = false;

    El tercer paso seria realizar un ciclo while, cada vez que el perceptron falle la variable $epocas aumentara, esto para determinar el numero de epocas. Cambiamos el valor de la variable $ciclo para que se siga repitiendo. Declaramos cada uno de los pesos con un numero aleatorio(Eso se podria simplificar usando una funcion). La variable $umbral que se trata de un peso sinaptico mas tambien sera rendomizada.

    while(!$ciclo){
    $epocas++;
    $ciclo = true;
    $peso1 = rand(1,10000000);
    $peso1 = ($peso1 - rand(1,10000000))/10000000;
    $peso2 = rand(1,10000000);
    $peso2 = ($peso2 - rand(1,10000000))/10000000;
    $umbral = rand(1,10000000);
    $umbral = ($umbral - rand(1,10000000))/10000000;
    ...

    El cuarto paso seria crear una funcion, tratando de hacer una representacion basica de una sigmoide a la cual la llamaremos "sigmoide".

    La funcion generaliza entre 2 numeros 0 y 1. Si el valor introducido es mayor a 0 la funcion retorna un 1 y un 0 en caso contrario.

    function sigmoide($valor){
    return $variable = ($valor > 0) ? 1 : 0;
}

    Lo siguiente seria crear la funcion principal del perceptron la "neurona". Se trata de la suma de las entradas multiplicado por los pesos mas el umbral.

    function neurona($entrada1,$entrada2,$peso1,$peso2,$umbral){
    return $total=$umbral+$entrada1*$peso1+$entrada2*$peso2;
}

    Teniendo las 2 funciones creadas, creamos un ciclo for dentro del ciclo while despues de haber declarado las variables randomizadas.Este ciclo for se repetira por cada uno de set de datos(En este caso 4).Realizamos un IF en el cual incluiremos la funcion sigmoide y como parametro agregaremos la funcion neurona y condicionaremos si el resultado de la funcion sigmoide(Que debe retornar 1 o 0), es diferente del resultado deseado, si es verdad entonces seguimos iterando, para eso cambiamos la variable $ciclo a false. Cerramos el if, el for y el while.

    ...
for($i=0; $i❮$longitud; $i++){
    if(sigmoide(neurona($entradas[$i][0],$entradas[$i][1],$peso1,$peso2,$umbral))!=$entradas[$i][2]){
        $ciclo = false;
        }
    }
}

    Al final colocamos echos para visualizar los resultados.

    echo"Peso 1: ". $peso1."\n";

echo "Entrada 1: 0 Entrada 2: 1 : " . sigmoide(neurona($entradas[2][0],$entradas[2][1],$peso1,$peso2,$umbral))."\n";
